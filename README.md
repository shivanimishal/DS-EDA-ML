## 🗺️ 1. India Live Interactive Maps

A dynamic, real-time geospatial visualization project tracking key environmental and meteorological metrics across India.

* **High-Precision Boundaries:** Utilizes highly accurate geo-json boundary data to map India's states and territories flawlessly.

## 🌍 2. Global GDP: Current & Projections

An in-depth macroeconomic analysis project visualizing the world's economic health, comparing current output with future growth expectations.

### ✨ Top Features
* **Current vs. Forecast:** Contrasts 2024 Latest Actuals with 2031 IMF GDP Growth Rate Forecasts.
* **Interactive Toggling:** Users can easily toggle between Total GDP (Log Scale), GDP Per Capita, and Growth Forecasts via interactive UI buttons.
* **Top 15 Economies Analysis:** Includes a deep-dive analytical view into the performance, share, and growth vectors of the world's top 15 largest economies.
* **Advanced Color Mapping:** Uses intuitive color scales (e.g., diverging scales for growth where Green = Growing, Red = Shrinking) to make complex global data instantly understandable.
